## Installation

1. **Clone the repository:**

   ```sh
   git clone https://github.com/Leigham/WebsiteLayout.git
   ```

2. **Navigate to the project directory:**

   ```sh
   cd WebsiteLayout
   ```

3. **Run the Python server:**
   ```sh
    python3 .\startserver.py
   ```
